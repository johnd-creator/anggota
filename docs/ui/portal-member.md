# Portal Anggota UX

## Header & Layout
- Header: foto/avatar bundar, nama, badge status, unit
- Indicator kelengkapan data dengan progress bar, CTA "Lengkapi Profil"
- Layout dua kolom di desktop, stack di mobile

## Edit Flow
- Panel slide-over untuk edit alamat, telepon, foto
- Status pending ditampilkan jika menunggu persetujuan Admin Unit
- Change History ringkas: status terakhir dan catatan

## Timeline
- Vertikal dengan ikon/warna per event (mutasi, cuti, reaktivasi)
- Filter tahun & pencarian cepat

## Kartu Digital
- Card responsif: background brand, foto bundar, QR code, chip status
- Varian state: Aktif, Cuti, Suspended, Resign/Pensiun
- Tap to flip (opsional) untuk detail tambahan
- Preview + tombol Unduh PDF + Bagikan tautan verifikasi

## Verifikasi QR
- Summary card dengan status indicator dan pesan yang jelas
- Timestamp scan dan pesan keamanan
- CTA Hubungi Serikat bila tidak aktif

## Notification Center
- Panel notifikasi dengan ikon, warna, aksi cepat
- Empty state yang ramah

## Aksesibilitas
- Kontras warna badge dan QR page sesuai WCAG AA
- Focus ring pada elemen interaktif
- Responsif untuk timeline, kartu, dan slide-over
